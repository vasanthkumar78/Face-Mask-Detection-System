# Real-Time Face Mask Detection System

## Overview
This project is a Real-Time Face Mask Detection System developed using Python, TensorFlow, Keras and OpenCV.

The system detects whether a person is wearing a face mask or not through a live webcam feed.

## Features
- Real-time face detection
- Mask / No Mask classification
- CNN-based model
- OpenCV integration

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy

## Project Workflow
1. Collect dataset
2. Preprocess images
3. Train CNN model
4. Save trained model
5. Real-time detection using webcam

## Applications
- Hospitals
- Airports
- Railway Stations
- Public Safety Monitoring
